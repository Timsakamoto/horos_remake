# Project Antigravity

**Target**: macOS用DICOMビューア「Horos」の完全なる再構築（Reborn）。
**Philosophy**: HorosのUI/UX、操作感、機能を100%再現する。ただし、内部コードは完全にモダンな技術に置き換える。

## The Antigravity Stack
- **Core**: Electron + React + TypeScript
- **Rendering**: Cornerstone3D
- **DICOM Logic**: dcmjs
- **Database**: RxDB
- **Styling**: Tailwind CSS

## Usage

### Install Dependencies
```bash
npm install
```

### Run Development Server
```bash
npm run dev
```

### Build for Production
```bash
npm run build
```
